## Getting Started


### Prerequisites

This project uses npm as package manager



### Run Locally

Clone the project

```bash
  git clone https://github.com/Lamarr99/Project_Bank_webpage.git
```

Go to the project directory

```bash
  cd app
```

Install dependencies

```bash
  npm install `only tailwind`
```

Start the server

```bash
  npm run dev
```
